# INSEAD 26D Elective Planner

Interactive tool for planning your INSEAD MBA elective courses across P3, P4, and P5.

## Features
- Configure your campus for each period (FBL, SGP, ABD, Exchange)
- Browse 195 courses with descriptions, ratings, and seat counts
- Priority weighting sliders to rank courses by your interests
- Live credit and constraint checking
- Shortlist courses you're considering before committing
- Bidding strategy helper with point allocation
- Selections saved in your browser (private to you)

## How to Use
1. Visit the planner link
2. Set your campus for each period
3. Adjust priority sliders to match your career interests
4. Browse and add courses to each period
5. Check constraints are met (credits, CDT requirement)
6. Plan your bidding points for P3

Built by Isabella Isotta. Course data from INSEAD MBA Office.
